# is2dArray <Lang dart js />

지정된 배열이 2차원 배열이면 `true`를 반환합니다.

## Parameters

- `array::any[]`

## Returns

> boolean

## Examples

```javascript [JavaScript]
is2dArray([1]); // Returns false
is2dArray([[1], [2]]); // Returns true
```

```dart [Dart]
is2dArray([1]); // Returns false
is2dArray([[1], [2]]); // Returns true
```

:::
