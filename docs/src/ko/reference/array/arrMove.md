# arrMove <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

배열의 특정 요소의 위치를 지정된 위치로 이동합니다. (위치는 0부터 시작합니다.)

## Parameters

- `array::any[]`
- `from::number`
- `to::number`

## Returns

> any[]

## Examples

```javascript
arrMove([1, 2, 3, 4], 1, 0); // Returns [2, 1, 3, 4]
```
