# arrTo1dArray <Lang dart js />

다차원 배열의 모든 요소를 1차원 배열로 병합합니다.

## Parameters

- `array::any[]`

## Returns

> any[]

## Examples

::: code-group

```javascript [JavaScript]
arrTo1dArray([1, 2, [3, 4]], 5); // Returns [1, 2, 3, 4, 5]
```

```dart [Dart]
arrTo1dArray([1, 2, [3, 4]], 5); // Returns [1, 2, 3, 4, 5]
```

:::
