# numUnique <Lang dart js />

타임스탬프와 난수를 결합한 고유의 숫자를 반환합니다.

이 함수는 충돌 가능성을 가지고 있습니다. 대규모 데이터를 처리하거나 고유 ID를 부여할 경우 주의해야합니다.

## Parameters

No parameters required

## Returns

> number

## Examples

::: code-group

```javascript [JavaScript]
numUnique(); // Returns `177052123219057200`
```

```dart [Dart]
numUnique(); // Returns `177052123219057200`
```

:::
