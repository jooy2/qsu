# encodeBase64 <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

<span class="node-required">Node.js 런타임 필요 ('qsu/node')</span>

지정된 문자열을 Base64로 인코딩합니다.

## Parameters

- `str::string`

## Returns

> string

## Examples

```javascript
encodeBase64('this is test'); // Returns 'dGhpcyBpcyB0ZXN0'
```
