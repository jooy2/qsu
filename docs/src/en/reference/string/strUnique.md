# strUnique <Lang dart js python />

Remove duplicate characters from a given string and output only one.

## Parameters

- `str::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
strUnique('aaabbbcc'); // Returns 'abc'
```

```dart [Dart]
strUnique('aaabbbcc'); // Returns 'abc'
```

```python [Python]
strUnique('aaabbbcc')  # Returns 'abc'
```

:::
