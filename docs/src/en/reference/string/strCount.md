# strCount <Lang dart js />

Returns the number of times the second String argument is contained in the first String argument.

## Parameters

- `str::string`
- `search::string`

## Returns

> number

## Examples

::: code-group

```javascript [JavaScript]
strCount('abcabc', 'a'); // Returns 2
```

```dart [Dart]
strCount('abcabc', 'a'); // Returns 2
```

:::
