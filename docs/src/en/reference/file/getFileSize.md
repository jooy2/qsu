# getFileSize <Lang js dart />

<NodeRequired en />

Calculates the size of the file at the given path.

## Parameters

- `filePath::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
getFileSize('/home/user/test.txt'); // 300
```

```dart [Dart]

getFileSize('/home/user/test.txt'); // 300
```

:::
