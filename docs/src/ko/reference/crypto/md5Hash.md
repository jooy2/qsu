# md5Hash <Lang dart js />

<NodeRequired ko />

문자열 데이터를 md5 해시 값으로 변환하여 반환합니다.

## Parameters

- `str::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
md5Hash('test'); // Returns '098f6bcd4621d373cade4e832627b4f6'
```

```dart [Dart]
md5Hash('test'); // Returns '098f6bcd4621d373cade4e832627b4f6'
```

:::
