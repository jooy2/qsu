# mul <Lang js dart python />

Returns after multiplying all n arguments of numbers or the values of a single array of numbers.

## Parameters

- `numbers::...number[]`

## Returns

> number

## Examples

::: code-group

```javascript [JavaScript]
mul(1, 2, 3); // Returns 6
mul([1, 2, 3, 4]); // Returns 24
```

```python [Python]
mul(1, 2, 3) # Returns 6
mul([1, 2, 3, 4]) # Returns 24
```

:::
