# getRamSize <Lang js python />

<NodeRequired ko />

현재 장치의 전체 RAM 크기를 가져옵니다. 단위가 포함된 읽기 쉬운 텍스트로 반환됩니다. 소수점은 올림됩니다.

## Parameters

필수 매개 변수 없음

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
console.log(getRamSize()); // Returns '8 GB'
```

```python [Python]
print(getRamSize())  # Returns '8 GB'
```

:::
