# len <Lang dart js />

Returns the length of any type of data. If the argument value is `null` or `undefined`, `0` is returned.

## Parameters

- `data::any`

## Returns

> boolean

## Examples

```javascript [JavaScript]
len('12345'); // Returns 5
len([1, 2, 3]); // Returns 3
```

```dart [Dart]
len('12345'); // Returns 5
len([1, 2, 3]); // Returns 3
```

:::
