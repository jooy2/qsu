# sha1Hash <Lang dart js />

<NodeRequired en />

Converts String data to sha1 hash value and returns it.

## Parameters

- `str::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
sha1Hash('test'); // Returns 'a94a8fe5ccb19ba61c4c0873d391e987982fbbd3'
```

```dart [Dart]
sha1Hash('test'); // Returns 'a94a8fe5ccb19ba61c4c0873d391e987982fbbd3'
```

:::
