# arrWithNumber <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

시작...끝 값의 순서로 배열을 생성하고 반환합니다.

## Parameters

- `start::number`
- `end::number`

## Returns

> number[]

## Examples

```javascript
arrWithNumber(1, 3); // Returns [1, 2, 3]
arrWithNumber(0, 3); // Returns [0, 1, 2, 3]
```
