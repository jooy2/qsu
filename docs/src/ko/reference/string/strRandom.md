# strRandom <Lang dart js />

주어진 길이의 숫자 또는 대문자와 소문자를 포함하는 임의의 문자열을 반환합니다. 기본 반환 길이는 12입니다.

## Parameters

- `length::number`
- `additionalCharacters::string?` <DartNamed />

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
strRandom(5); // Returns 'CHy2M'
```

```dart [Dart]
strRandom(5); // Returns 'CHy2M'
```

:::
