# arrShuffle <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

Shuffle the order of the given array and return.

## Parameters

- `array::any[]`

## Returns

> any[]

## Examples

::: code-group

```javascript [JavaScript]
arrShuffle([1, 2, 3, 4]); // Returns [4, 2, 3, 1]
```

```dart [Dart]
arrShuffle([1, 2, 3, 4]); // Returns [4, 2, 3, 1]
```

:::
