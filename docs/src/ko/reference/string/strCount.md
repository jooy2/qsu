# strCount <Lang dart js />

첫 번째 String 인자에 두 번째 String 인자가 포함된 횟수를 반환합니다.

## Parameters

- `str::string`
- `search::string`

## Returns

> number

## Examples

::: code-group

```javascript [JavaScript]
strCount('abcabc', 'a'); // Returns 2
```

```dart [Dart]
strCount('abcabc', 'a'); // Returns 2
```

:::
