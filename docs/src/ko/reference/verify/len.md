# len <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

Returns the length of any type of data. If the argument value is `null` or `undefined`, `0` is returned.

## Parameters

- `data::any`

## Returns

> boolean

## Examples

```javascript
len('12345'); // Returns 5
len([1, 2, 3]); // Returns 3
```
