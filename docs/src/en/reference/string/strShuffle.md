# strShuffle <Lang dart js python />

Randomly shuffles the received string and returns it.

## Parameters

- `str::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
strShuffle('abcdefg'); // Returns 'bgafced'
```

```dart [Dart]
strShuffle('abcdefg'); // Returns 'bgafced'
```

```python [Python]
strShuffle('abcdefg')  # Returns 'bgafced'
```

:::
