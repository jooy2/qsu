# numPick <Lang dart js />

min과 max값 사이 중 임의의 숫자를 선택하여 리턴합니다.

## Parameters

- `min::number`
- `max::number`

## Returns

> number

## Examples

::: code-group

```javascript [JavaScript]
numPick(1, 5); // Returns 1~5
numPick(10, 20); // Returns 10~20
```

```dart [Dart]
numPick(1, 5); // Returns 1~5
numPick(10, 20); // Returns 10~20
```

:::
