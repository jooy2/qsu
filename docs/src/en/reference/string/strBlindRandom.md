# strBlindRandom <Lang js />

Replace strings at random locations with a specified number of characters (default 1) with characters (default \*).

## Parameters

- `str::string`
- `blindLength::number`
- `blindStr::string || '*'`

## Returns

> string

## Examples

```javascript
strBlindRandom('hello', 2, '#'); // Returns '#el#o'
```
