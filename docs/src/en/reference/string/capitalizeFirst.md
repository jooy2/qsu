# capitalizeFirst <Lang dart js />

Converts the first letter of the entire string to uppercase and returns.

## Parameters

- `str::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
capitalizeFirst('abcd'); // Returns 'Abcd'
```

```dart [Dart]
capitalizeFirst('abcd'); // Returns 'Abcd'
```

:::
