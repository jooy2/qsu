# isEmail <Lang dart js />

주어진 인수 값이 유효한 이메일인지 확인합니다.

## Parameters

- `email::string`
- `onlyLowerCase::boolean || false` <span class="named">Dart:Named</span>

## Returns

> boolean

## Examples

```javascript
isEmail('abc@def.com'); // Returns true
isEmail('ABC@def.com', true); // Returns false
```
