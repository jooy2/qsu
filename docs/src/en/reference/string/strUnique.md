# strUnique <Lang dart js />

Remove duplicate characters from a given string and output only one.

## Parameters

- `str::string`

## Returns

> string

## Examples

::: code-group

```javascript [JavaScript]
strUnique('aaabbbcc'); // Returns 'abc'
```

```dart [Dart]
strUnique('aaabbbcc'); // Returns 'abc'
```

:::
