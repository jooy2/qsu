# arrCount <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

지정된 배열의 각 고유 값에 대한 중복 횟수를 반환합니다. 배열 값은 `String` 또는 `Number` 유형만 가능합니다.

## Parameters

- `array::string[]|number[]`
- `count::number`

## Returns

> object

## Examples

```javascript
arrCount(['a', 'a', 'a', 'b', 'c', 'b', 'a', 'd']); // Returns { a: 4, b: 2, c: 1, d: 1 }
```
