# is2dArray <Lang dart js python />

Returns `true` if the given array is a two-dimensional array.

## Parameters

- `array::any[]`

## Returns

> boolean

## Examples

::: code-group

```javascript [JavaScript]
is2dArray([1]); // Returns false
is2dArray([[1], [2]]); // Returns true
```

```dart [Dart]
is2dArray([1]); // Returns false
is2dArray([[1], [2]]); // Returns true
```

```python [Python]
is2dArray([1])  # Returns False
is2dArray([[1], [2]])  # Returns True
```

:::
