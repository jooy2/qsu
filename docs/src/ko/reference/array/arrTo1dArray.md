# arrTo1dArray <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

Merges all elements of a multidimensional array into a one-dimensional array.

## Parameters

- `array::any[]`

## Returns

> any[]

## Examples

```javascript
arrTo1dArray([1, 2, [3, 4]], 5); // Returns [1, 2, 3, 4, 5]
```
