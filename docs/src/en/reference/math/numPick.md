# numPick <Lang dart js />

Returns a randomly selected number between the min and max values.

## Parameters

- `min::number`
- `max::number`

## Returns

> number

## Examples

::: code-group

```javascript [JavaScript]
numPick(1, 5); // Returns 1~5
numPick(10, 20); // Returns 10~20
```

```dart [Dart]
numPick(1, 5); // Returns 1~5
numPick(10, 20); // Returns 10~20
```

:::
