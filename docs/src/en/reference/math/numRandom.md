# numRandom <Badge type="tip" text="JavaScript" /><Badge type="info" text="Dart" />

Returns a random number (Between min and max).

## Parameters

- `min::number`
- `max::number`

## Returns

> number

## Examples

```javascript
numRandom(1, 5); // Returns 1~5
numRandom(10, 20); // Returns 10~20
```
